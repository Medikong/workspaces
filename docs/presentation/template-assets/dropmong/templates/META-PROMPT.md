# 단일 페이지 템플릿 메타프롬프트

```text
역할:
DropMong 실무 프로젝트용 16:9 발표 템플릿 설계자

입력:
- DESIGN.md
- template-plan.json
- 대상 이미지와 같은 이름의 .prompt.json
- template_id
- template_name
- page_purpose
- audience_takeaway
- dominant_visual_type
- required_labels
- example_data
- semantic_color_role

목표:
컴포넌트 모음이 아닌 하나의 완성된 발표 페이지 이미지 제작
청중이 제목과 중심 시각물을 3초 안에 구분할 수 있는 구성

출력:
- 1600×900 PNG
- 한 파일에 한 페이지
- PNG와 같은 이름의 `.prompt.json`
- 상단 DropMong 로고 표시
- 마스코트는 표지, 장 전환과 마무리 페이지에만 표시
- 하단에 프로젝트명, 발표자명, 날짜를 넣을 footer 공간
- 출처 영역과 페이지 번호 없음

디자인:
- 배경 #FCFCFD
- 제목 #111827
- 중심색 #6C3DF5
- 보조색 #B69BFF
- 확인 정보 #2563EB
- 기준 충족 #15803D
- 결정 필요 #B45309
- 기준 초과 #DC2626
- Pretendard 계열 글꼴
- 좌우 80px, 위 56px, 아래 48px 안전 여백
- 1px 구분선, 2px 도식선, 8·12·16px 모서리
- 그림자 없음 또는 매우 약한 그림자 한 곳

구성:
- 제목 최대 2줄
- 본문 중심 시각물의 화면 점유율 60~75%
- 보조 설명 최대 2개
- 독립 카드 최대 2개
- 제목 → 중심 시각물 → 핵심 수치 → footer 순서
- 시퀀스 다이어그램은 참여자 2~5개, 메시지 4~8개
- 차트는 축, 단위, 수치와 데이터 상태 표시

수치:
- 기본 수치는 `000 ms`, `00.0%`, `000 RPS`, `00m` 형태의 자리 표시 사용
- 실제 결과가 아닌 수치는 `예시 데이터 · 실제 측정값 아님` 표기
- 이전 HPA 수치는 별도의 이전 실험 근거 템플릿에서만 사용
- 우열을 암시하지 않는 템플릿에는 동일하거나 중립적인 값 사용

언어:
- 설명은 자연스러운 한국어
- KEDA, HPA, Kafka, event, consumer lag, p99 같은 기술 용어는 원문 유지
- 번역체 표현, 긴 문단, 명사 연속 나열 제외

레퍼런스:
- 다크 테마 이미지는 제목 크기, 중심 시각물 비율, 그래프와 수치 배치만 참고
- 검은 배경, 네온 색상, 글로우와 원본 도형 복제 금지
- DropMong 브랜드 에셋은 색, 로고와 마스코트 형태 참고

금지:
- 축소 템플릿 모음
- 컴포넌트 카탈로그
- 동일한 카드 4개 이상
- 대시보드 형태
- 상품 UI
- 임의의 연구 결과
- 출처 footer
- 워터마크
```

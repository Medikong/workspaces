# 발표 템플릿 이미지 에셋

이 폴더는 특정 발표 도구에 묶이지 않는 공통 이미지 레퍼런스를 보관합니다. PowerPoint, Slidev, Reveal.js에서 같은 디자인 기준을 참고할 수 있도록 브랜드별 하위 폴더로 구분합니다.

## 에셋 목록

| 폴더 | 내용 |
| --- | --- |
| `dropmong/` | DropMong 발표 디자인 토큰, 컴포넌트와 템플릿 시트 |

각 이미지는 완성된 발표 페이지가 아니라 화면을 설계할 때 참고하는 시트입니다. 실제 발표에서는 필요한 구성만 골라 텍스트, 표, 차트와 도형을 편집 가능한 요소로 다시 만듭니다.
